Usage
-----

```
cmf restore [options] <packagePath>
```

### Arguments

Name | Description
---- | -----------
`<packagePath>` | Package path

### Options

Name | Description
---- | -----------
`-r, --repo, --repos <repos>` | Repositories where dependencies are located (folder)
`-?, -h, --help` | Show help and usage information
